firmware

Flash your buddy.

Firmware is bundled. Connect your ESP32 over USB, hit flash. No esptool install, no manual file hunting. Replace firmware/buddy.bin to update.

requires chrome or edge 89+
before you start
  • [1] Chrome or Edge 89+ — Web Serial API is required. Firefox does not support it.
  • [2] USB cable — data cable, not charge-only. Connect directly to your ESP32 or dev board.
  • [3] Firmware is pre-loaded — bundled from firmware/buddy.bin. Replace that file to update. You can also override it by dropping a different .bin onto the zone.
  • [4] Close other serial monitors — Arduino IDE serial monitor, PlatformIO terminal, or any other tool holding the port.

bootloader mode

Hold BOOT while pressing RESET to enter bootloader mode manually. Most dev boards handle this automatically via DTR/RTS.

firmware patch block

Add this struct to your ESP32 firmware. The flash tool finds the magic bytes and writes your settings directly into the binary. 

struct __attribute__((packed)) {
  uint8_t magic[8] =
    {0xBD,0xBD,0xBD,0xBD,
     0x42,0x55,0x44,0x59};
  char ssid[32] = "YOUR_SSID";
  char pass[64] = "YOUR_PASS";
  char host[40] = "192.168.1.x";
  uint32_t port = 3000;
  char id[16]   = "BDY-00001";
} buddy_cfg;
